# 📖 Proje Hakkında

**Kaç Yaktı?**, araç sahiplerinin yakıt tüketimlerini kolayca takip edebilmesi amacıyla geliştirilmiş açık kaynaklı bir web uygulamasıdır.

Uygulama sayesinde;

- Yakıt alımlarınızı kaydedebilirsiniz.
- Yakıt tüketiminizi analiz edebilirsiniz.
- Kilometre başına maliyetinizi görebilirsiniz.
- Toplam harcamalarınızı takip edebilirsiniz.
- Aracınızın yakıt geçmişini saklayabilirsiniz.

Tüm veriler tarayıcı üzerinde saklanır ve herhangi bir sunucu gerektirmez.
